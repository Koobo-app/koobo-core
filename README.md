# Koobo-Core
## Usage

```bash
create a .env file using the .env.sample as an example
```

```bash
npm install
```

```bash
npm run dev
```

## Endpoints
```bash
POST /api/v1/signup
```

```bash
POST /api/v1/login    // Gives back a token
```

```bash
GET /api/v1/welcome   // send a token in the header with the x-access-token key
```
