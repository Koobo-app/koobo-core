# Koobo-Core
## Usage

```bash
create a .env file using the .env.sample as an example
```

```bash
npm install
```

```bash
npm run dev
```

## Endpoints
```bash
POST /register
```


```bash
POST /login    // Gives back a token
```



```bash
GET /welcome   // send a token in the header with the x-access-token key
```
